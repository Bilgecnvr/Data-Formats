# Data-Formats
This repository was created for data formats homework.
 
A personnel object has been modeled in the homework and displayed in **XML**, **YAML**, and **JSON** formats.

 `id` (integer)   
 `ad` (string)   
 `soyad` (string)   
 `pozisyon` (string)   
 `maas` (float) 
 `adres` (string)
 `ise_giris_tarihi` (string, YYYY-MM-DD)
 `aktif_mi` (boolean)
